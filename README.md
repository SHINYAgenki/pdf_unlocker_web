# PDF Unlock on WebAPP

パスワード保護されたPDFファイルをブラウザ上で手軽に解除できるWebアプリケーションです。

## 概要
業務で頻繁に発生する「パスワード付きPDFの解除作業」を効率化するために開発しました。
機密性の高いファイルを外部のオンラインサービスにアップロードするリスクを避け、セキュアな環境（ローカルや自社サーバー）で動作させることを目的としています。

## 技術スタック
- **Language:** Python 3.x
- **Framework:** Django 5.0.4
- **Library:** PyPDF2 (PDF操作・暗号化解除)
- **Frontend:** HTML5, CSS3 (Responsive Design)

## こだわったポイント
1. **メモリ内処理によるセキュリティ向上**
   アップロードされたファイルをサーバーのディスクに一時保存せず、`io.BytesIO`を用いてメモリ上で処理・返却しています。これにより、サーバー内に機密データが残留するリスクを物理的に排除しました。
2. **マルチバイトファイル名への対応**
   日本語ファイル名が含まれる場合でもダウンロード時に文字化けしないよう、`urllib.parse.quote`を使用して適切にエンコーディングを行っています。
3. **直感的なユーザー体験**
   Djangoのフォーム機能を活用し、ファイル選択から解除、ダウンロードまでをシームレスに行えるインターフェースを実装しました。
